## Getting Started

First, run the development server:

```bash
npm run dev
```

Node: v22.14.0

Setup:

1. clone the repo.
2. run npm i with using node 22
3. start the backend at port 4000
4. start the app using npm run dev

UI will run on default port 3000
Backend must be runing at port 4000.
